# 🎨 Stylish Calculator UI with CI/CD Deployment

---

# 🧠 Project Overview

This project is a **modern frontend calculator** built using HTML, CSS, and JavaScript.

It is automatically deployed using GitHub Actions to GitHub Pages.

---

# 📦 Project Structure

```
projects/calculator-ui/
│
├── index.html
├── style.css
├── script.js
```

---

# ⚙️ Features

* Basic arithmetic operations
* Modern UI with gradients and animations
* Responsive layout
* Interactive buttons

---

# 🔄 CI/CD Workflow

```text
Code Push → GitHub Actions → Deploy to GitHub Pages → Live Website
```

---

# 🌐 Deployment

The application is deployed using GitHub Pages.

After each push:

* GitHub Actions triggers deployment
* Site is updated automatically

---

# ⚙️ Pipeline Explanation

Pipeline file: `.github/workflows/frontend.yml`

Steps:

1. Checkout code
2. Deploy using GitHub Pages action
